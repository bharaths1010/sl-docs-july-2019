=======================
Levels of Testing
=======================

- Unit
- Integration
- Regression Testing
- System
- Acceptance (Also known as UAT, typically takes place in Pre-Prod)

TDD (Test Drive Development)
  - Write the test cases (before you write your code)
  - Run the tests and see them fail
  - Write / Modify your code to make the tests pass
  - Refactor and repeat

BDD (Behavior Driven Development)
  - Emphasis is on Collaboration
3 Amigos
  - Product Owner
  - Developers
  - Testers
